Running this app locally.

As an alternative to running this app on the SciLifeLab Serve servers you can run it on your own computer. This is possible because all apps on our service are packaged as Docker containers. Docker containers include everything that is needed to run a particular piece of software and are easy to run on any machine.

Step 1. Install Docker

You will need Docker Engine or Docker Desktop running on your computer. Follow official instructions for your operating system to do that.

Step 2. Download the Docker image of this app

Navigate to Terminal (or Windows Terminal) window and run the following command in order to download the Docker image of this app.

docker pull ...

If the image was successfully downloaded you show now be able to see it in the list of images by running the following command.

docker image ls

Step 3. Launch the app

You can now launch the app by running the following command in your Terminal (or Windows Terminal) window.

docker run ...

Navigate to .... to open the app.
